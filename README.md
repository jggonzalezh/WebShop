## running the Backend

Build the .NET Core project   using the .NET Core CLI, which is installed with [the .NET Core SDK](https://www.microsoft.com/net/download). Then run
these commands from the CLI in the directory of any sample:

```console
dotnet build
dotnet run
```

These will install any needed dependencies, build the project, and run
the project respectively.


##  running the FrontEnd

Run the React project using npm

Clone the project

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
